# Support

Use the project repository for issues, discussions, and code review:

- GitHub: [autobrr/upbrr](https://github.com/autobrr/upbrr)
- Issues: [github.com/autobrr/upbrr/issues](https://github.com/autobrr/upbrr/issues)
- Community: [autobrr Discord](https://discord.autobrr.com)

When reporting a problem, include the command or workflow, relevant tracker names, sanitized logs, and whether the same behavior occurs in CLI, GUI, or web mode.

Never include credentials, tokens, passkeys, cookies, full secret-bearing payloads, or private tracker-sensitive data in public reports.
