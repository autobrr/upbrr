# API docs

The embedded web server exposes the checked OpenAPI spec and Swagger UI.

Start the built-in web server:

```bash
upbrr serve
```

Open Swagger UI in a browser:

```text
/api/docs
```

Use the OpenAPI spec directly as JSON or YAML:

```text
/api/openapi.json
/api/openapi.yaml
```

The published docs site exposes the same checked spec at:

```text
/openapi.yaml
```
