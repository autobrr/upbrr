# Descriptions

Descriptions are built from tracker profiles, metadata, mediainfo, screenshots, and configured formatting.

The description builder can:

- fetch generated description groups
- render BBCode previews
- save per-group overrides
- keep tracker-specific description behavior separate from shared content

Use description overrides for intentional release-specific edits. Prefer fixing shared builder behavior when the generated output is broadly wrong.
